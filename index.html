Get started locally

Using Thunder locally is a great way to learn how it works and prototype analyses. You just need to have Scala (2.10.3) and Python (2.7.6) (with NumPy and SciPy; you can get all these Python tools at once for free with Anaconda).

Here's how to get started in four steps.

1) Download the latest versions of Spark and Thunder and set some paths.

export PYTHONPATH=your/path/to/thunder/python/:$PYTHONPATH
export SPARK_HOME=your/path/to/spark

2) Build Spark (this will take a few minutes) 

cd $SPARK_HOME
sbt/sbt assembly

3) Start the PySpark shell (if you have iPython, also set IPYTHON=1)

$SPARK_HOME/bin/pyspark

4) Try an analysis 

>> from thunder.io import DataSets
>> from thunder.factorization import PCA
>> data = DataSets.load(sc, "iris")
>> model = PCA(k=4).fit(data.rdd)

Get started on EC2

In just a few minutes you can be using Thunder on Amazon's EC2.

1) Sign up for an AWS account, generate a key pair, and use your Amazon EC2 access key ID and secret access key (where to find these) to set environmental variables.

export AWS_ACCESS_KEY_ID=your_access_key_id
export AWS_SECRET_ACCESS_KEY=your_secret_access_key

2) Download the latests versions of Spark and Thunder and set a path.

export SPARK_HOME=your/path/to/spark

3) Run the EC2 launch script from inside the Thunder folder (will take a few minutes).

helper/ec2/thunder-ec2 -k keyname -i keyname.pem -s N_NODES launch CLUSTERNAME

where keyname and keyname.pem are from the key pair you generated, N_NODES is the number of nodes you want, and CLUSTERNAME is a name to call your cluster


4) Log into your newly configured cluster

helper/ec2/thunder-ec2 -k keyname -i keyname.pem login CLUSTERNAME

5) Try an analysis 

pyspark
>> from thunder.io import DataSets
>> from thunder.factorization import PCA
>> data = DataSets.load(sc, "iris")
>> model = PCA(k=4).fit(data.rdd)
