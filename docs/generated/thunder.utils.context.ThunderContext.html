importStackAsBlocks(datafile, dims, nblocks=None, filerange=None)[source]¶

Convert data from binary stack files to blocks of an RDD, which can either be saved to flat binary files, or returned as an flattened RDD (see convertStack and importStack)

Stacks are typically flat binary files containing 2-dimensional or 3-dimensional image data

We assume there are multiple files:

file0.stack, file1.stack, file2.stack, ...

This function loads the same contiguous block from all files, and rewrites the result to flat binary files of the form:

block0.bin, block1.bin, block2.bin, ...
