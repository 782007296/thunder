thunder.Registration¶

class thunder.Registration¶

Class for constructing registration algorithms.

Construct a registration algorithm by specifying its name, and passing any additional keyword arguments. The algorithm can then be used to fit registration parameters on an Images object.

Parameters:

method : string

A registration method, options include ‘crosscorr’ and ‘planarcrosscorr’
