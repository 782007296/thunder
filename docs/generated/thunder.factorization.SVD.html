thunder.factorization.SVD¶

class thunder.factorization.SVD(k=3, method='direct', maxiter=20, tol=1e-05)¶

Large-scale singular value decomposiiton on a dense matrix represented as an RDD or RowMatrix with nrows and ncols

Parameters:

k : int, optional, default = 3

Number of singular vectors to estimate

method : string, optional, default “direct”

Whether to use a direct or iterative method

maxiter : int, optional, default = 20

Maximum number of iterations if using an iterative method

tol : float, optional, default = 0.00001

Tolerance for convergence of iterative algorithm

Attributes

u (RDD of nrows (tuple, array) pairs, each array of shape (k,)) Left singular vectors
s (array, shape(nrows,)) Singular values
v (array, shape (k, ncols)) Right singular vectors
