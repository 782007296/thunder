thunder.decoding.uniclassify.MassUnivariateClassifier¶

class thunder.decoding.uniclassify.MassUnivariateClassifier(paramfile)[source]¶

Base class for mass univariate classification. Assumes that for each signal, each time point belongs to one of several categories. Uses independent classifiers to train and predict. Example usage: determining how well each individual neural signal predicts a behavior.

Parameters:

paramfile : str or dict

A MAT file or Python dictionary containing parameters for training classifiers. At minimum must contain a “labels” field, with the label the classify at each time point. Can additionally include fields for “features” (which feature was present at each time point) and “samples” (which sample was present at each time point)

Attributes

labels (array) Array containing the label for each time point
features (array) Which feature was present at each time point
nfeatures (int, optional) Number of features
samples (array) Which sample does each time point belong to
sampleids (list) Unique samples
nsamples (int) Number of samples
