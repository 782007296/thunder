Time Series¶

timeseries.fourier.Fourier(freq) Class for computing fourier transform
timeseries.stats.Stats(statistic) Class for computing simple summary statistics
timeseries.query.Query(indsfile) Class for querying key-value data by averaging together records with the specified indices (typically, the keys are spatial coordinates and the values are time series, so this returns the center coordinates and average time series of the specified records).

timeseries.localcorr.LocalCorr([neighborhood]) Class for computing local correlations
